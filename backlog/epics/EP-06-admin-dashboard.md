# EP-06: Panel de Administración (Admin Dashboard)

**Estado:** Backlog
**Prioridad:** Alta

## Descripción

Interfaz de administración para gestionar socios, visualizar métricas clave del club, administrar roles y generar reportes de uso, pagos y reservas.

## Objetivo de Negocio

Dar al equipo directivo visibilidad completa del estado operativo del club y control sobre usuarios y configuraciones, sin necesidad de acceso directo a la base de datos.

## Criterios de Completitud del Epic

- [ ] El admin puede ver la lista completa de socios con estado y plan de membresía.
- [ ] El admin puede activar, desactivar o cambiar el rol de un socio.
- [ ] El dashboard muestra métricas clave: socios activos, reservas del día, ingresos del mes.
- [ ] El admin puede exportar reportes de socios y pagos en CSV.
- [ ] El manager tiene acceso a una vista reducida del dashboard (sin gestión de roles ni finanzas).
