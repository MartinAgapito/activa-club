# EP-07: Notificaciones (Notifications)

**Estado:** Backlog
**Prioridad:** Media

## Descripción

Sistema de notificaciones basado en AWS SNS para alertar a los socios sobre eventos relevantes: confirmaciones de reserva, recordatorios de pago, vencimiento de membresía y nuevas promociones.

## Objetivo de Negocio

Mantener a los socios informados de forma proactiva para reducir cancelaciones por olvido, mejorar la experiencia y aumentar el uso de las instalaciones.

## Criterios de Completitud del Epic

- [ ] Los socios pueden suscribirse y desuscribirse de notificaciones por email.
- [ ] Se envía notificación de confirmación al crear una reserva.
- [ ] Se envía recordatorio 24 horas antes de una reserva.
- [ ] Se envía alerta cuando el pago de membresía falla o está próximo a vencer.
- [ ] El admin puede enviar una notificación masiva a todos los socios activos.
