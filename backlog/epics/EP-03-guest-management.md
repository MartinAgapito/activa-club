# EP-03: Gestión de Huéspedes (Guest Management)

**Estado:** Backlog
**Prioridad:** Media

## Descripción

Los socios pueden registrar invitados para que accedan temporalmente a las instalaciones. Se generan códigos de acceso, se controla la cantidad de ingresos y se registra el historial de visitas.

## Objetivo de Negocio

Permitir que los socios traigan invitados de forma controlada, generando trazabilidad de accesos y limitando el uso por socio según su plan de membresía.

## Criterios de Completitud del Epic

- [ ] Un socio puede registrar un invitado con nombre y datos básicos.
- [ ] Se genera un código de acceso único y temporal para el invitado.
- [ ] El sistema limita la cantidad de invitados por socio según su plan de membresía.
- [ ] El staff puede validar el código de acceso en la entrada.
- [ ] El historial de visitas de invitados es consultable por el socio y el admin.
